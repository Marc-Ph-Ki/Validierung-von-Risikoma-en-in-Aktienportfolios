# Validierung-von-Risikoma-en-in-Aktienportfolios
In diesem Projekt setze ich mich kritisch mit der Genauigkeit gängiger Risikokennzahlen zur Bewertung von Aktienportfolios auseinander. Ziel der Untersuchung ist es, die Verlässlichkeit des modellbasierten Value-at-Risk (VaR) gegenüber historischen Simulationsansätzen zu validieren. Im Rahmen der Analyse untersuche ich reale Markteffekte wie Volatilitätscluster und "Fat Tails", die oft zu einer Unterschätzung systemischer Risiken führen. Zur statistischen Absicherung der Ergebnisse verwende ich Backtesting-Verfahren wie den Kupiec-Test sowie t-Statistiken. Die Analyse zeigt signifikante Abweichungen (p-Wert < 0,05) auf und belegt, dass rein modellbasierte Ansätze in Stressphasen dazu neigen, das tatsächliche Risiko zu unterschätzen, was weitreichende Implikationen für die Kapitalunterlegung in Finanzinstituten hat.
